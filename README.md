Healthcare Operations Dashboard

Overview

This Tableau dashboard analyzes healthcare operational performance using patient visit data. The dashboard provides insights into patient volume, wait times, appointment completion rates, and readmission trends.

CSV Dataset
GitHub

Key Metrics
Total Patients: 1,200
Average Wait Time: 36.28 Minutes
Readmission Cases: 93
Completed Appointments: 972

Visualizations

Readmission Analysis

Business Value

This dashboard helps healthcare administrators monitor operational efficiency, patient flow, and quality-of-care indicators. The visualizations highlight departmental performance and support data-driven decision-making.
